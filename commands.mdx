## Command Prefix

Bender uses the `!` prefix by default. You can change this in the dashboard.

## Command Categories

Bender has commands in the following categories:

- **Economy** — Currency, gambling, businesses
- **Leveling** — XP, levels, leaderboards
- **Moderation** — Kick, ban, mute, warnings
- **Utility** — Server info, user info, polls
- **Fun** — Memes, games, giveaways

## Using Slash Commands

Bender also supports Discord slash commands. Type `/` in any channel to see all available commands.
