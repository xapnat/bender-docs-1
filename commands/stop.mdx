## Usage

```
!stop
```

## Description

Stops any currently running game or activity in the channel. If you have an active game session (e.g. Blackjack, Mines), this command will end it immediately.

**Note:** Stopping a game in progress will forfeit your bet.

## Examples

```
!stop
```
