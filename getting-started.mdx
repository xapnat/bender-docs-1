## Add Bender to your Server

Invite Bender to your Discord server using the invite link on [bender.best](https://bender.best).

## First Steps

Once Bender is in your server, you can start using commands right away. Head over to the [Commands](/docs/commands) section to see all available commands.

## Dashboard

You can configure Bender through the web dashboard at [bender.best/dashboard](https://bender.best/dashboard). The dashboard lets you customize:

- Leveling settings
- Economy settings
- Auto-moderation
- Custom embeds
- And much more

Join our [Support Server](https://discord.gg/bender) if you have any questions.
