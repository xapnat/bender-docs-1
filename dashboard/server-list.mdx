The **Server List** page helps you present your server publicly and manage the feedback and edits that come with that visibility.

## Listing settings

Use the settings tab to control how your server appears on the directory.

- Publish or hide your listing.
- Show or hide activity indicators like message and voice stats.
- Write a server description up to 2,000 characters.
- Add up to two tags, such as Community or Gaming, and suggest new tags when needed.
- Select one or more server languages, including English, Deutsch, Espanol, Francais, Portugues, Turkce, Russian, Japanese, Korean, or Chinese.
- Set or automatically generate a permanent invite link.
- Define who can leave reviews, such as everyone, server members only, or boosters.
- Set a minimum membership time for reviews.
- Add up to five bump reward tiers that link bump counts to reward roles.

## Reviews

The reviews tab helps you monitor public feedback about your server.

- Read star ratings and written feedback.
- Check report counts when a review may need attention.
- Review the author, rating, report count, and date for each review.
- Open full review details with **View**.
- Reply to or report reviews from the review modal.

<Image src="/assets/images/dashboard/server-list-reviews.png" alt="Reviews Page" size="sm"/>

## Changes

Changes make listing updates easier to audit over time.

- Compare old and new values for edited fields.
- Review property names such as description when listing fields change.
- Check pending, approved, and rejected statuses.
- Review when a change happened.
- Approve or reject updates if your workflow uses review controls.

<Image src="/assets/images/dashboard/server-list-changes.png" alt="Changes Page" size="sm"/>

<Callout type="tip">
  A polished description, accurate tags, and a permanent invite link usually matter more than constant listing edits. Keep the basics clean first.
</Callout>
