The **Community** section groups together the features that help your server feel active, rewarding, and fun to come back to.

## Engagement systems

<Accordion>
  <AccordionItem title="Leveling">
    Track text and voice XP with toggles for public leaderboard, delete on leave, text XP, voice XP, ignore muted users, and ignore deafened users. You can set XP per message, XP per minute, channel/role/user exceptions for text and voice XP, role bonus multipliers, reward roles, message cooldowns, temporary XP boosts, level-up message templates, and rank card templates.
  </AccordionItem>
  <AccordionItem title="Rewards">
    Reward members for promoting the server through vanity status text or guild tags. Each reward type can be enabled separately and supports trigger text, alert message templates, reward roles, and role or user exceptions.
  </AccordionItem>
  <AccordionItem title="Giveaways">
    Review active, draft, and ended giveaways. Entries show title, winner count, participants, and end time, with actions to reroll winners, edit, or delete. Use Create Giveaway to start a new giveaway.
  </AccordionItem>
  <AccordionItem title="Birthdays">
    Enable birthday announcements, choose the announcement channel, set the announcement time, select templates with or without age display, and assign temporary birthday roles.
  </AccordionItem>
  <AccordionItem title="Booster Roles">
    Enable booster roles, choose the base role, set the server limit, allow or restrict sharing with friends, set shared role and member limits, and manage allowed roles, allowed users, and blacklisted words. The Booster Roles tab lists existing custom booster roles.
  </AccordionItem>
  <AccordionItem title="Auto Roles">
    Automatically assign roles to users, bots, everyone, or boosters. Join-based sections support up to 10 roles, booster sections support up to 5 roles, and role connections let you assign roles conditionally based on existing membership.
  </AccordionItem>
</Accordion>

## Utility and retention tools

<Accordion>
  <AccordionItem title="Counters">
    Create up to three live channel counters that display stats such as member count. When none are configured, the page shows an empty state.
  </AccordionItem>
  <AccordionItem title="Starboard">
    Highlight popular messages in a dedicated channel. The editor supports a board name, starboard channel, reaction threshold, trigger emojis, embed template, and behavior toggles such as ignore self-stars, auto-react, remove on unstar, ignore bot messages, allow multiple reactions, remove self-star reactions, auto-react first only, remove on delete, and remove bot reactions.
  </AccordionItem>
  <AccordionItem title="Suggestions">
    Create a suggestion type and review submitted suggestions. The dashboard currently limits setup to one suggestion type.
  </AccordionItem>
  <AccordionItem title="Temp Voice">
    Create temporary voice hubs that spin up channels when members join a base channel. The hub editor controls hub name, base channel, category, channel name template, user limit, bitrate, keep-alive time, ownership lock, permission sync, default role, owner permissions, interface template, exceptions, moderator lists, and blacklisted words.
  </AccordionItem>
  <AccordionItem title="Music">
    Configure default volume, autoplay, DJ roles, and exception channels from Settings. The Player tab lets staff search and play songs or playlists, reload, stop, shuffle, repeat, apply filters, and review the queue.
  </AccordionItem>
</Accordion>

<Callout type="tip">
  If your server is still small, start with Leveling, Auto Roles, and Suggestions first. They add visible value quickly without a lot of setup overhead.
</Callout>
