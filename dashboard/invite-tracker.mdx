**Invite Tracker** shows who is bringing new members into your server and helps you reward genuine growth.

## Invites

The invites view lists invite usage so you can see who invited whom and how many successful joins each inviter has.

- Search for specific invite codes or members.
- Review inviter performance without checking Discord invite links manually.
- Spot suspicious patterns when invites rise but valid joins stay low.
- Use the empty state as confirmation when no invite entries have been collected yet.

<Image src="/assets/images/dashboard/invite-tracker-invites.png" alt="Invites Page" size="sm"/>

## Settings

Use the settings tab to define how invite tracking should behave.

- Enable or disable tracking for the server.
- Set a **fake threshold** so very new accounts do not count toward rewards.
- Turn on logging, pick a channel for invite events, and choose the log template.
- Create up to 10 invite milestone rewards.
- Decide whether invite rewards should stack when a member qualifies for multiple milestones.

<Callout type="info">
  A fake threshold is one of the most important settings on this page. It helps keep reward systems focused on real growth instead of throwaway accounts.
</Callout>

## Leaderboard

The leaderboard ranks members by invite performance so you can spotlight your most effective promoters.

- Sort members by invite-related metrics.
- Search for a specific user.
- Run **Sync Roles** when you need to refresh invite-based rewards.
- Use the empty state to confirm that leaderboard data has not been collected yet.
