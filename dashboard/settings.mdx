Use **Settings** to control how your server appears in the dashboard and how the bot behaves at a basic server-wide level.

## Branding

The customization area controls the bot's profile.

- Upload a custom icon for the bot profile.
- Change the bot display name for your server.
- Edit the **About Me** text used on the profile card.
- Set a banner image to match your server branding.
- Use the live preview to confirm the profile before saving.

<Image src="/assets/images/dashboard/settings-1.png" alt="Settings Page" size="sm"/>

## Server Configuration

This section covers the core defaults used across your server.

- **Language** changes the default language used in supported messages and interfaces. The dashboard supports English and Deutsch.
- **Timezone** uses an IANA timezone, such as `Europe/Berlin` or `Australia/Lord_Howe`, so logs and scheduled actions use the right local time.
- **Command Prefix** defines how members trigger text commands when a prefix is required. The default prefix is `!`.
