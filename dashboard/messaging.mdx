## Messages

- Manage welcome, goodbye, and custom messages from one place.
- Review cards for welcome, goodbye, boost, or custom message types.
- See whether each message targets a channel or DMs.
- Expand a message card to change the channel and template used.
- Preview each embed before sending it live.
- Search existing messages when your server uses multiple announcements or onboarding flows.
- Delete messages that are no longer used or create a new one with **New Message**.

## Embeds

Embeds are reusable templates for polished announcements.

- Store your own templates for repeat use.
- Copy default templates to save setup time.
- Browse community templates when you need ideas or faster starting points.
- Review tags such as category and version on each template card.
- Preview, edit, duplicate, copy, or delete templates from the card actions.
- Use the Embed Editor to set template name, category, thread name, auto-delete timer, components such as buttons or dropdowns, and live preview content.
- Reference the variable list for personalization data such as user, member, thread, and moderation variables.

## Automations

Automations connect triggers and actions so routine workflows happen automatically.

- Build IF/THEN flows from dashboard events.
- Turn automations on or off without deleting them.
- Review how many automations your server is currently using compared with its limit.
- Edit an automation in the visual flow builder.
- Add trigger blocks, IF conditions, THEN actions, ELSE IF branches, and ELSE branches.
- Copy or delete automations from their row actions.
