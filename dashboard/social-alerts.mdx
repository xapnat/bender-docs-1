**Social Alerts** keeps your server connected to external content by posting updates from creators and cleaning up shared social links when needed.

## Creator alerts

<Accordion>
  <AccordionItem title="YouTube">
    Add up to three channels to receive upload notifications. Each entry lets you choose the Discord notification channel, select a notification template, and enable alerts for videos or Shorts.
  </AccordionItem>
  <AccordionItem title="Twitch">
    Use Notifications mode for go-live alerts with a Discord channel and template. Use Repost mode to repost Twitch links automatically with controls for enabled state, delete original, reply to message, suppress embed, and trigger word filtering.
  </AccordionItem>
  <AccordionItem title="TikTok">
    Configure notifications with a Discord channel and template, or use Repost mode with the same delete original, reply, suppress embed, and trigger word options.
  </AccordionItem>
  <AccordionItem title="Kick">
    Add Kick channels for go-live notifications. When none are configured, the page prompts you to add a channel.
  </AccordionItem>
</Accordion>

## Link reposting

<Accordion>
  <AccordionItem title="Instagram">
    Repost Instagram content automatically, delete the original link message if needed, reply to the original message, suppress the native Discord embed, or require a trigger word before reposting.
  </AccordionItem>
  <AccordionItem title="X (Twitter)">
    Configure repost behavior for X links with the same quality-of-life options used for Instagram, including trigger words and embed suppression.
  </AccordionItem>
</Accordion>
